# 🟡 Resume Intelligence

> **AI-Powered ATS Resume Analyzer built using Python, Streamlit & Google Gemini AI**

---

## 📌 Overview

Resume Intelligence is an AI-powered resume analysis platform that helps job seekers evaluate their resumes against a target job role.

The application analyzes resumes using Google's Gemini AI and provides professional hiring insights including ATS score estimation, strengths, weaknesses, missing skills, interview questions, career roadmap, and hiring manager feedback.

---

## ✨ Features

- 📄 PDF Resume Upload
- 🤖 AI Resume Analysis
- 📊 ATS Resume Evaluation
- 💪 Strength & Weakness Detection
- 🧠 Skill Gap Analysis
- 🎯 Missing Skills Recommendation
- 💼 Hiring Manager Feedback

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Koushik-mangalagiri/resume-intelligence.git
```

Go into the project

```bash
cd resume-intelligence
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
streamlit run app.py
```

---

## 💻 How It Works

1. Upload Resume (PDF)
2. Enter Target Job Role
3. Click Analyze Resume
4. Gemini AI analyzes the resume
5. View detailed ATS Report
6. Review Interview Questions
7. Improve Resume
